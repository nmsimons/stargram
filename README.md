# ⭐ Stargram

A React app that reads data from Excel files and generates printable output.

## Features

- **Excel Upload** — Drag & drop or click to upload `.xlsx` / `.xls` files
- **Data Preview** — View parsed spreadsheet data in a table
- **Printable Output** — Print-optimized view with one-click printing

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- [React](https://react.dev/) + [Vite](https://vite.dev/)
- [SheetJS (xlsx)](https://sheetjs.com/) — Excel parsing
- [react-dropzone](https://react-dropzone.js.org/) — File upload UX
